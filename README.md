# MemoryPJ

写真付きの思い出を記録・管理するためのWebアプリです。

## 概要

このアプリは、日常の出来事や旅行などの思い出を「写真＋メモ」として記録し、あとから振り返れるようにすることを目的として作成しました。

単なるメモではなく、視覚的に思い出を残せるようにすることで、記録の楽しさと振り返りやすさを重視しています。

## 使用技術

* Python 3.12
* Django 5.1
* PostgreSQL
* Git / GitHub

## 主な機能

* 思い出一覧表示（写真付き）
* 思い出詳細表示
* 思い出の投稿（画像＋テキスト）
* 編集・削除機能（CRUD）
